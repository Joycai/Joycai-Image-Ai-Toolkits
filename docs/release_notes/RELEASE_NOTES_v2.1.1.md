# Release Notes - v2.1.1 (The Refinement & Optimization Update)

This release focuses on improving the Workbench user experience, refactoring internal prompt architectures, and enhancing project documentation with live version indicators.

## 🎨 Workbench & UX Improvements
*   **AI Rename Architecture**: Refactored the internal prompt architecture for AI-powered renaming to be more robust and extensible.
*   **Enhanced Workbench UI**: Minor UI refinements in the Workbench screen for better workflow efficiency and clearer feedback.
*   **Localization Sync**: Synchronized and updated localization strings across all supported languages (English, Chinese, Japanese, Traditional Chinese).

## 🛠️ Infrastructure & Maintenance
*   **Dependency Updates**: Upgraded `google_fonts` to `8.0.2` for improved typography rendering and stability.
*   **Build Automation**: Introduced new internal automation for build count synchronization across Android, iOS, macOS, and Windows, ensuring version consistency.
*   **Documentation Badges**: Added live version and Flutter version badges to `README.md` and `README_CN.md` for real-time project status visibility.

## ⚙️ Core Enhancements (from v2.1.x cycle)
*   **Event-Driven Task Queue**: Improved task management with an event-driven architecture for more reliable batch processing.
*   **Model Streaming Config**: Enhanced configuration options for model response streaming.

*v2.1.1 continues our commitment to providing a polished and stable AI toolkit. Thank you for your continued support!*
